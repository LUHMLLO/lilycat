## Button <small class="clr-subtext txt-nm txt-lowercase"> `<button/>` </small>

Buttons in web development allow users to perform actions or navigate to other pages.
They come in various styles to suit different needs and are effective for prompting
users to take action to progress within an interface.

<grid class='grid cols-2 outln outln-primary radius-3xs'>
  ```html
  <button>
    <span>button</span>
  </button>
  ```
  <column class='flex place-center p-5xl'>
    <button>
      <span>button</span>
    </button>
  </column>
</grid>

### -- parentElement <small class="clr-subtext txt-sm"> PROPS </small>

<ly-slider tmpl-expand>
  | prop        | description                      | requires |
  | ----------- | -------------------------------- | -------- |
  | - - outline | sets outline style               | -        |
  | - - fab     | changes style to look like a FAB | -        |
</ly-slider>

<grid class="grid cols-2 g-sm">
    <column class="grid outln outln-primary radius-3xs">
        ```html
        <button data-props="--outline">
          <span>button</span>
        </button>
        ```

        <div class="flex place-center p-5xl">
            <button data-props='--outline'>
              <span>button</span>
            </button>
        </div>

    </column>

    <column class="grid outln outln-primary radius-3xs">
        ```html
        <button data-props="--fab">
          <span>button</span>
        </button>
        ```

        <div class="flex place-center p-5xl">
            <button data-props='--fab'><span>button</span></button>
        </div>

    </column>

</grid>

### -- childrenElements <small class="clr-subtext txt-sm"> PROPS </small>

<ly-slider tmpl-expand>
  | prop      | description            | required for                   |
  | --------- | ---------------------- | ------------------------------ |
  | - - lead  | set’s as leading item  | left side optical corrections  |
  | - - trail | set’s as trailing item | right side optical corrections |
</ly-slider>

<grid class="grid cols-2 g-sm">
    <column class="grid outln outln-primary radius-3xs">
