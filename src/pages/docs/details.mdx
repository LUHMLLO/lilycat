## Details / Summary <small class="clr-subtext txt-nm txt-lowercase"> native </small>

The `<summary>` element provides a summary or caption for the content of a `<details>` element.
It is used to toggle the visibility of the details content when clicked, making it ideal for
presenting collapsible sections of information or controls in a user interface.

<grid class='grid cols-2 outln outln-primary radius-3xs'>
  ```html
  <details>
    <summary>
      <span>summary</span>
    </summary>
    . . . content here . . .
  </details>
  ```
  <column class='align-center[items] flex place-center p-5xl'>
    <details>
      <summary>
        <span>summary</span>
      </summary>
      . . . content here . . .
    </details>
  </column>
</grid>

### -- summaryElement <small class="clr-subtext txt-sm"> PROPS </small>

| prop        | description                         | requires   |
| ----------- | ----------------------------------- | ---------- |
| - - button  | changes style to look like a button | -          |
| - - outline | sets outline style                  | - - button |
| - - fab     | changes style to look like a FAB    | - - button |
