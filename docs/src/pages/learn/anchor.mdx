## Anchor <small>native</small>

<br />

`<a/>` elements in web development create hyperlinks, which are used to navigate to other pages, sections within a page, or external resources. They can be styled to look like buttons or other interactive elements, making them versatile for navigation and linking purposes. Anchors are essential for creating a connected and interactive web experience.

<br />

<div data-is="grid" style="--cols: 2; --gap: var(--nm);">
  <div data-is="group" style="gap: var(--sm);">
     Anchor elements without a [href] will visually behave as a paragraph.

    <div data-is="codeblock">
      <div data-part="example">
        <a>
          <span>anchor</span>
        </a>
      </div>

      ```html
      <a>
        <span>anchor</span>
      </a>
      ```
    </div>

  </div>

  <div data-is="group" style="gap: var(--sm);">
    When it has an `href` attribute it behaves like a normal hyperlink.

    <div data-is="codeblock">
      <div data-part="example">
        <a href="#href">
          <span>anchor</span>
        </a>
      </div>

      ```html
      <a href="#href">
        <span>anchor</span>
      </a>
      ```
    </div>

  </div>

  <div data-is="group" style="gap: var(--sm);">
    If you like to keep it discrete but want to ensure it's accessible when using `[href]`, you can use the `data-variant="text"` attribute to preserve hyperlink interactions.

    <div data-is="codeblock">
      <div data-part="example">
        <a href="#" data-variant="text">
          <span>anchor</span>
        </a>
      </div>

      ```html
      <a href="#" data-variant="text">
        <span>anchor</span>
      </a>
      ```
    </div>

  </div>
</div>
