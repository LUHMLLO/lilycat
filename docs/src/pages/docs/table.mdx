## Table <small class="clr-subtext txt-nm txt-lowercase"> native </small>

`<table/>` element in web development is used to create a structured 
grid layout for displaying tabular data. It consists of one or more 
`<tr/>` (table row) elements, each containing one or more `<td/>` (table data) 
or `<th/>` (table header) elements. Tables are essential for presenting 
data in a structured and organized manner.

<grid class='grid cols-2 outln outln-primary radius-3xs'>
  ```html
  <table>
    <thead>
      <tr>
        <th>Name</th>
        <th>Age</th>
        <th>City</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>John Doe</td>
        <td>30</td>
        <td>New York</td>
      </tr>
      <tr>
        <td>Jane Smith</td>
        <td>25</td>
        <td>Los Angeles</td>
      </tr>
    </tbody>
  </table>
  ```
