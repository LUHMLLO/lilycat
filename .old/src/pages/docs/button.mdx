## Button <small class="clr-subtext txt-nm txt-lowercase"> native </small>

`<button/>` in web development allow users to perform actions or navigate to other pages.
They come in various styles to suit different needs and are effective for prompting
users to take action to progress within an interface.

<grid data-props="--2" class='outln outln-primary radius-3xs'>
  ```html
  <button>
    <span>button</span>
  </button>
  ```
  <column class='align-center[items] flex place-center p-5xl'>
    <button>
      <span>button</span>
    </button>
  </column>
</grid>

### -- parentElement <small class="clr-subtext txt-sm"> PROPS </small>

| prop        | description                      | requires |
| ----------- | -------------------------------- | -------- |
| - - outline | sets outline style               | -        |
| - - fab     | changes style to look like a FAB | -        |

<grid data-props="--2" class="g-sm">
    <column class="grid outln outln-primary radius-3xs">
        ```html
        <button data-props="--outline">
          <span>button</span>
        </button>
        ```

        <div class="flex place-center p-5xl">
            <button data-props='--outline'>
              <span>button</span>
            </button>
        </div>

    </column>

    <column class="grid outln outln-primary radius-3xs">
        ```html
        <button data-props="--outline --fab">
          <span>button</span>
        </button>
        ```

        <div class="flex place-center p-5xl">
            <button data-props='--outline --fab'><span>button</span></button>
        </div>

    </column>

</grid>

### -- childrenElements <small class="clr-subtext txt-sm"> PROPS </small>

| prop      | description            | purpose                        |
| --------- | ---------------------- | ------------------------------ |
| - - lead  | set’s as leading item  | left side optical corrections  |
| - - trail | set’s as trailing item | right side optical corrections |

<grid data-props="--2" class="g-sm">

        <button data-props="--outline">
